# 🌱 Agrizen – AI Powered Agriculture

Agrizen is an AI-powered smart agriculture platform designed to help farmers make informed decisions through intelligent crop recommendations, disease detection, weather insights, and modern farming solutions. Built with Next.js, React, TypeScript, Firebase, and Generative AI, the platform provides a responsive and user-friendly experience for sustainable agriculture.

---

## 🚀 Features

- 🌾 AI-powered crop recommendations
- 🦠 Crop disease detection
- 🌦️ Weather insights for better farming decisions
- 🔐 Secure user authentication with Firebase
- 📊 Interactive dashboard
- 📱 Responsive and modern user interface
- 🤖 Generative AI-powered farming assistance

---

## 🛠️ Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase
- Generative AI
- Node.js
